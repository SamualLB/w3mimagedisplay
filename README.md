# w3mimagedisplay

Tiny Crystal library for using w3mimagedisplay

## Installation

1. Add the dependency to your `shard.yml`:
```yaml
dependencies:
  w3m_image_display:
    github: SamualLB/w3mimagedisplay
```
2. Run `shards install`
